# Bee Rescue Saga
Project for the POOIG class 2020-2021


## Installation

Dans une console UNIX, compiler le projet avec :  ```./gradlew build```

Puis lancer l'installation avec : ```./gradlew run```

## Usage

### CLI

```./gradlew run --args='cli' ```

### GUI

```./gradlew run --args='gui' ```
